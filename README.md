# 💥위험한 직장인들

위험하지만 치열하게 공부하는 직장인들의 스터디 관리 공간입니다.  
본 저장소는 스터디 전체 현황을 모니터링하고, 자동화된 랭킹 시스템을 운영합니다.



## 🏆 주간 명예의 전당 (Weekly Leaderboard)

우리 스터디는 꾸준한 학습을 권장하기 위해 **자동 랭킹 시스템**을 운영 중입니다.

* **업데이트**: 매주 월요일 오전 09:00 (KST)
* **집계 대상**: 조직 내 모든 저장소의 커밋 내역
* **결과 확인**: 슬랙 `#weekly-report` 채널에서 🥇, 🥈, 🥉 메달 주인공을 확인할 수 있습니다.


## 🗺️ 스터디 아지트

모여서 각자 코딩 하고 싶을 경우에는 ZEP 스페이스에 모여서 진행합니다.  
스페이스 접속/종료 시 슬랙 `#zep-notification` 채널에 접속 알림 메시지가 전송됩니다.

* **ZEP**: [[danger](https://zep.us/play/9PXO7g)] (비밀번호는 방장에게 문의..)


## 📜 스터디 규칙

즐겁고 지속 가능한 성장을 위해 아래 규칙을 준수합시다.

1. **리포지토리는 각자 생성하기**: 자유롭게 저장소를 만들어 각자 공부한 내용을 커밋하세요.
2. **꾸준하게 공부하기**: 어떤 사소한 내용이라도 좋으니 자신의 저장소에 공부한 내용을 커밋하세요.
